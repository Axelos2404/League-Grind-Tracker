League Grind Tracker ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how your information is handled when you use our desktop application.

1. Information We Collect
- Local Game Data: The application interacts with your locally running League of Legends client (127.0.0.1) to read active game states, summoner profiles, and gameflow phases. This data is processed entirely on your local machine.
- Public Riot API Data: The application fetches public data from Riot Games APIs, including your Riot ID (gameName and tagLine), ranked tiers, LP, and recent match histories.
- Local Storage / Preferences: Preferences such as accent color, background opacity, and session persistence settings are stored locally on your device via browser local storage. We do not transmit or store this data on external servers.

2. How We Use Information
- To display your live ranked session statistics, match history, and performance metrics directly inside the application overlay.

3. Data Sharing
- We do not collect, share, or sell any personal data. All API interactions occur directly between your device and Riot Games' official API servers.

4. Changes to This Policy
- We may update this Privacy Policy from time to time. Any changes will be posted within our GitHub repository.
